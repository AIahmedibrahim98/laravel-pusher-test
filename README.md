### This is a Test project for Pusher Realtime

## how to use

### 1. install

```
composer install
```

### 2. create .env file

```
cp .env.example .env
```

### 3. generate key

```
php artisan key:generate
```

### 4. set your pusher key in .env file

### 5. migrate database

```
php artisan migrate
```

### 6. run server

```
php artisan serve
```
